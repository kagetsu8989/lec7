# Backup-ийн төрлүүд

# Full Backup 
- Бүх өгөгдлийг бүтнээр нь хуулж авна
- Давуу тал: сэргээхэд амар
- Сул тал: их зай эзэлнэ

# Incremental Backup
- Сүүлд backup хийснээс хойш өөрчлөгдсөн хэсгийг хадгална
- Давуу тал: хурдан, бага зай эзэлнэ
- Сул тал: сэргээхэд олон файл хэрэгтэй

# Differential Backup
- Эхний full backup-аас хойш өөрчлөгдсөн бүх мэдээллийг халгална
- Давуу тал: incremental-аас сэргээхэд амар 
- Сул тал: зай арай их эзэлнэ

# 2дох асуулт
# Backup Strategy

Backup strategy гэдэг нь өгөгдлийг алдагдахаас хамгаалах зорилгоор
ямар аргаар, хэдийд, хаана нөөцлөхийг тодорхойлсон төлөвлөгөө юм.

# Backup хийх давтамж
- Daily (өдөр бүр)
- Weekly (долоо хоног бүр)
- Monthly (сар бүр)

# 1. Full Backup
- Бүх өгөгдлийг бүтнээр нь хадгална
- Сэргээхэд хамгийн хялбар

# 2. Incremental Backup
- Сүүлд хийсэн backup-аас хойш өөрчлөгдсөн хэсгийг хадгална
- Хурдан, бага зай эзэлнэ

# 3. Differential Backup
- Эхний full backup-аас хойш өөрчлөгдсөн бүх өгөгдлийг хадгална
- Сэргээхэд incremental-аас амар

# Хадгалах байршил
- Local storage (компьютер, external disk)
- Cloud storage (Google Drive, Dropbox)
