# Backup-ийн төрлүүд

# Full Backup 
- Бүх өгөгдлийг бүтнээр нь хуулж авна
- Давуу тал: сэргээхэд амар
- Сул тал: их зай эзэлнэ

# Incremental Backup
- Сүүлд backup хийснээс хойш өөрчлөгдсөн хэсгийг хадгална
- Давуу тал: хурдан, бага зай эзэлнэ
- Сул тал: сэргээхэд олон файл хэрэгтэй

# Differential Backup
- Эхний full backup-аас хойш өөрчлөгдсөн бүх мэдээллийг халгална
- Давуу тал: incremental-аас сэргээхэд амар 
- Сул тал: зай арай их эзэлнэ
